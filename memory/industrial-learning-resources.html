快速上手路径（按顺序，总共 1~2 小时）

  1. 装 ModbusPal → 定义 3 个 HoldingRegister → 跑 ModbusDriver.ReadAsync → 亲眼看到 ushort[]
  2. 装 MQTTX → 连接 test.mosquitto.org → 写一段代码把刚才读到的寄存器值 Publish 到云端主题
  3. 装 Prosys OPC UA → 浏览它的 NodeId 树 → 理解 ns=3;s=Temperature 为什么长这样
  4. 回头看代码 → 打开你的 IProtocolDriver → 打开 ModbusDriver → 明白每一个设计是为什么
