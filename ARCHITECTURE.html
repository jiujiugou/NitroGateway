什么是"闭环"

  1. 网关启动 → Scheduler 每 1 秒调一次采集
  2. 内置 模拟 PLC 在 127.0.0.1:502 返回 3 个传感器值（自动波动）
  3. ModbusTcpDriver 读到 ushort[] → 解码为 double/int → RawPointValue
  4. Pipeline 做缩放 + 死区 → PointSnapshot
  5. Dispatcher 双写：measurements 表 + forward_buffer 表
  6. HealthReporter → DeviceHealthMonitor 计采集成败
  7. Scheduler 每 5 秒调一次转发 → Forwarder JSON 序列化 → MQTT 发布
